\end{center} \begin{abstract} Homology and cohomology are prevalent methods of exploring a topological space. Homology groups are abelian groups derived in order to analyze the non-triviality of a topological space, for example holes in the space. de Rham cohomology groups are derived from the differential $k$-forms on smooth manifolds, which are generalizations of smooth functions. The relationship between homology and de Rham cohomology is not immediately clear by their construction. de Rham's Theorem proves that there is an isomorphism between the $k^{\text{th}}$ de Rham cohomology group and the group of homomorphisms from the $k^{\text{th}}$ homology group into $\mathbb{R}$. This paper explores the fundamentals of cubical homology and de Rham cohomology, and proves de Rham's theorem. \end{abstract} \tableofcontents \section{Homology}\label{sec:Hom} We formulate our understanding of homology using cubical homology, which begins with the definition of a cube. \begin{df} A \textit{k-cube} in a topological space $X$ is a continuous map $\Gamma: I^k \rightarrow X$, where $I^k$ is the k-dimensional cube, \[I^k = [0, 1] \times ... \times [0, 1] \subset \mathbb{R}^k\] A cube is \textit{degenerate} if, for some $i$, $\Gamma(t_1, ..., t_k)$ does not depend on $t_i$. %When $k=1$, this is equivalent to saying that the path $[0, 1] \rightarrow X$ is a constant path. \end{df} Let $\mathcal{Q}_k(X)$ denote the free abelian group generated by all $k$-cubes in $X$, and let $\mathcal{D}_k(X)$ denote the (free) subgroup generated by degenerate $k$-cubes. Let $\mathcal{C}_k(X)$ be the quotient group $\mathcal{Q}_k(X)/\mathcal{D}_k(X)$, called the group of (cubical) \textit{$k$-chains} on X. Therefore, \textit{$k$-chain} is a finite linear combination of non-degenerate $k$-cubes, $\Gamma = \sum_\alpha n_\alpha\Gamma_\alpha$ for integers $n_\alpha$. To motivate the definition of the boundary of a $k$-cube, consider a ($k$-1)-cube obtained by restricting the $i^{th}$ coordinate of a k-cube to $s$, where $s = \{0, 1\}$. We can refer to these as ``faces'' of the cube, denoted as $\partial_i^s\Gamma$. In particular, for a k-cube $\Gamma$, \begin{equation*}{\label{eq:bound1}} \partial_i^s(\Gamma) = \Gamma(t_1, \ldots, t_{i-1}, s, t_{i+1}, \ldots, t_k) \end{equation*} It is useful to introduce an additional function that defines the face of a cube, which will later be used in the definition of the orientation of a cube and Stokes Theorem for chains. We define $F^{k, i, s}: I^{k-1} \rightarrow I^k$ to be a diffeomorphism given by \begin{equation*}{\label{eq:bound2}} F^{k, i, s}(t_1, \ldots, t_{k-1}) = (t_1, \ldots, t_{i-1}, s, t_i, \ldots, t_{k-1}) \end{equation*} % \begin{tikzpicture} % % Draw the axes % \draw[->] (-1,0) -- (3,0) node[right] {$x$}; % \draw[->] (0,-1) -- (0,3) node[above] {$y$}; % % % Draw the square % \draw[pattern color=gray, pattern=crosshatch dots] (0,0) rectangle (2,2); % % % Label the vertices % \node[left] at (0,1) {$\partial_1^0$}; % \draw[->, blue] (0, 0) -- (0, 1) node[] {}; % \draw[blue] (0, 1) -- (0, 2); % \node[above] at (1,2) {$\partial_2^1$}; % \draw[->, blue] (0, 2) -- (1, 2) node[] {}; % \draw[blue] (1, 2) -- (2, 2); % \node[right] at (2,1) {$\partial_1^1$}; % \draw[->, blue] (2, 0) -- (2, 1) node[] {}; % \draw[blue] (2, 1) -- (2, 2); % \node[below] at (1,0) {$\partial_2^0$}; % \draw[->, blue] (0, 0) -- (1, 0) node[] {}; % \draw[blue] (1, 0) -- (2, 0); % % \draw[->] (5, 1) -- (8, 1) node[] {}; % % \draw[double distance=5mm, yscale=cos(70), xshift=13cm] (0:1) arc (0:180:1); % \draw[double distance=5mm, yscale=cos(70), xshift=13cm] (180:1) arc (180:360:1); % % \end{tikzpicture} \begin{df} The \textit{boundary} of a k-cube is defined as \[\partial\Gamma = \sum\limits_{i=1}^{k}(-1)^i(\partial_i^0\Gamma - \partial_i^1\Gamma) = \sum\limits_{i=1}^k (-1)^i (\Gamma \smallcirc F^{i, k, 0} - \Gamma \smallcirc F^{i, k, 1})\] \noindent The boundary of a k-chain $\Gamma \in \mathcal{C}_k(X)$ is extended linearly from the boundary of each term, so \[\partial(\sum n_\alpha\Gamma_\alpha) = \sum n_\alpha \partial\Gamma_\alpha\] \end{df} The boundary of a degenerate $k$-cube is a degenerate ($k$-$1$)-cube. So $\partial_k: \mathcal{C}_k(X) \rightarrow \mathcal{C}_{k-1}(X)$ is a homomorphism. \begin{thm} For any $k$+1-cube $\Gamma$, $\partial(\partial\Gamma)$ = 0. \end{thm} \begin{proof} Let $\Gamma$ be a basis element of $\mathcal{C}_{k+1}(X)$. Denote $\Gamma_{\widetilde{t_i}} = \partial_i^0\Gamma - \partial_i^1\Gamma$, and $\Gamma_{\widetilde{t_i}\widetilde{t_j}} = \partial_j^0\Gamma_{\widetilde{t_i}} - \partial_j^1\Gamma_{\widetilde{t_i}}$. We can see that \[\partial\Gamma = \sum\limits_{i=1}^k(-1)^i\Gamma_{\widetilde{t_i}}\]\[ \partial(\partial\Gamma) = \sum\limits_{ji}(-1)^i(-1)^{j-1}\Gamma_{\widetilde{t_i}\widetilde{t_j}}\] All the terms cancel, so $\partial(\partial\Gamma) = 0$. \end{proof} \noindent A k-chain is a \textit{k-cycle} if its boundary is 0. We define the following subgroups of $\mathcal{C}_k(X)$, where $\mathcal{C}_{-1}(X) = 0$: \[\mathcal{Z}_k(X) := \ker(\partial: \mathcal{C}_k(X) \rightarrow \mathcal{C}_{k-1}(X)) \text{ the set of all $k$-cycles}\] \[\mathcal{B}_k(X) := \im(\partial: \mathcal{C}_{k+1}(X) \rightarrow \mathcal{C}_k(X)) \text{ the set of boundaries}\] Notice $\mathcal{B}_k(X) \subseteq \mathcal{Z}_k(X)$ since $\partial \smallcirc \partial = 0$ \begin{df} For a topological space $X$, the $k^{th}$ cubical homology group is defined as \[H_k(X) = \mathcal{Z}_k(X)/\mathcal{B}_k(X)\] \end{df} \begin{ex} \textit{Calculating the homology of a point:} Let $p$ be the singleton set. Therefore, $\mathcal{C}_0(p) = \mathcal{Z}_0(p) = \mathbb{Z}$. $\mathcal{B}_0(p) = 0$ since $ \mathcal{C}_1(p) = 0$. Given a $1$-chain $\sum n_i\gamma_i$, where $\gamma_i:[0, 1] \rightarrow U$ is a continuous path with initial point $P_i$ and terminal point $Q_i$. The boundary of this chain is $\partial(\sum n_i \gamma_i) = \sum n_i(Q_i - P_i)$. Then, $\epsilon(\sum n_i(Q_i-P_i)) = 0$, since the inital and terminal points of a path must be in the same connected component. Therefore, $\mathcal{B}_0(X) \subseteq ker(\epsilon)$. Then, this cycle can be written as $\sum(Q_i - P_i)$ where $P_i$ and $Q_i$ are in the same component. Extending $f_\sharp$ linearly, $f_\sharp(\sum n_\alpha \Gamma_\alpha) = \sum n_\alpha f_\sharp (\Gamma_\alpha)$. By a simple calculation, $f_\sharp(\partial_i^s\Gamma) = f \smallcirc (\partial_i^s\Gamma) = \partial_i^s (f \smallcirc \Gamma)= \partial_i^s (f_\sharp \Gamma)$. Then, we can show $f_\sharp$ commutes with the boundary operator. \begin{align*} f_\sharp(\partial \Gamma) & = \sum\limits_{i=1}^k (-1)^i f_\sharp(\partial_i^0\Gamma - \partial_i^1\Gamma)\\ &= \sum\limits_{i=1}^k (-1)^i (\partial_i^0 (f_\sharp \Gamma) - \partial_i^1 (f_\sharp \Gamma))\\ &= \partial(f_\sharp \Gamma) \end{align*} If $\Gamma$ is a boundary of a $k+1$-chain, then $f_\sharp\Gamma$ is also a boundary. If $\Gamma$ is a cycle, then $\partial(f_\sharp\Gamma)) = f_\sharp(\partial\Gamma) = 0$, so $f_\sharp\Gamma$ is a cycle. Then, $f_\sharp$ induces a homomorphism $f_*: H_k(X) \rightarrow H_k(Y)$ that sends a homology class $[\Gamma]$ to $[f_\sharp \Gamma]$. \end{proof} \noindent \begin{rem} For continuous maps between topological spaces, $g:X \rightarrow Y$ and $f:Y \rightarrow Z$, $(f \smallcirc g)_* = f_* \smallcirc g_*$. Then, $\widetilde{\gamma} - \widetilde{\gamma}' = \partial(\widetilde{\beta})$ for some smooth (k+1)-chain $\widetilde{\beta}$, which implies \[\int_{\widetilde{\gamma}}\omega - \int_{\widetilde{\gamma'}}\omega = \int_{\partial\widetilde{\beta}}\omega = \int_{\widetilde{\beta}}d\omega = 0\] If $\omega = d\eta$ is exact, then \[\int_{\widetilde{\gamma}}\omega = \int_{\widetilde{\gamma}}d\eta = \int_{\partial\widetilde{\gamma}}\eta = 0\] \begin{lemma}\label{lemma:comm1} The following diagram commutes it $F: M \rightarrow N$ is a smooth map \begin{center} \begin{tikzcd} H^k(N) \arrow[r, "F^*"] \arrow[d, "\ell"] & H^k(M) \arrow[d, "\ell"]\\ Hom(H_k(N), \mathbb{R}) \arrow[r, "F^*"] & Hom(H_k(M), \mathbb{R}) \\ \end{tikzcd} \end{center} \end{lemma} \begin{proof} If $\Gamma \in C^\infty_k(M)$ and $\omega \in \Omega^p(N)$, \[\int_\Gamma F^*\omega = \int_{I^k} \Gamma^*F^*\omega = \int_{I^l}(F \circ \Gamma)^*\omega = \int_{F \circ \Gamma}\omega\] Then, $\ell(F^*[\omega])[\Gamma] = \ell[\omega][F \circ \Gamma] = \ell[\omega](F_*[\Gamma]) = F^*(\ell[\omega])[\Gamma]$ \end{proof} \begin{lemma}\label{lemma:comm2} The following diagram commutes for $M = U \cup V$, \begin{center} \begin{tikzcd} H^{k-1}(U \cap V) \arrow[r, "\delta"] \arrow[d, "\ell"] & H^k(M) \arrow[d, "\ell"]\\ Hom(H_{k-1}(U \cap V), \mathbb{R}) \arrow[r, "\partial^*"] & Hom(H_k(M), \mathbb{R}) \\ \end{tikzcd} \end{center} where $\delta$ is the connecting homomorphism of the Mayer-Vietoris sequence for de Rham cohomology, and $\partial^*(\gamma) = \gamma \circ \partial_*$, where $\partial_*$ is the connecting homomorphism for M-V sequence for homology. \end{lemma} \begin{proof} We want to show that $\ell(\delta[\omega])[\gamma] = (\partial^*\ell[\omega])[\gamma]$ for any $[\omega] \in H^{p-1}(U \cap V)$ and any $[\gamma] \in H_p(M)$. Let $\nu$ be a smooth $k$-form representing $\delta[\omega]$ and $c$ a smooth $(k-1)$-chain representing $\partial_*[\gamma]$. Then, it would suffice to show that $\int_\gamma \nu = \int_c \omega$. Let $c = \partial f$ where $f, f'$ are smooth k-chains in $U$ and $V$ such that $[f + f'] = [\gamma]$. Similarly, we can choose $\eta \in \Omega^{p-1}(U)$ and $\eta' \in \Omega^{p-1}(V)$ such that $\omega = \eta\big|_{U \cap V} - \eta\big|_{U \cap V}$ such that $\nu = d\eta$ on $U$ and $d\eta'$ on $V$. Then, because $\partial f + \partial f' = \partial \gamma = 0$ and $d\eta\big|_{U \cap V} - d\eta'\big|_{U \cap V} = d\omega = 0$, we have \begin{align*} \int_c\omega &= \int_{\partial f}\omega = \int_{\partial f} \eta - \int_{\partial f} \eta' = \int_{\partial f} \eta + \int_{\partial f'} \eta' \\ &= \int_{f} d\eta + \int_{f'} d\eta' = \int_f\nu + \int_{f'} \nu = \int_\gamma \nu \end{align*} \end{proof} \noindent In order to show that $\ell$ is an isomorphism, we will need the following lemmas: \begin{lemma} If $U$ is an open set in $\mathbb{R}^n$, then there is sequence of compact subsets $K_1, K_2, ...$ whose union is $U$, and so that $K_1 \subset Int(K_2) \subset K_2 \subset ... \subset K_n \subset Int(K_{n+1}) \subset ...$ \begin{proof} Start with a countable collection of open sets $U_i$ that cover $U$ such that $\bar{U_i}$ is compact and contained in $U$. For example, let each $U_i$ be a ball that is centered at a rational coordinate with a rational radii. Let $K_1 = \bar{U_1}$ and $K_2 = \bar{U_1} \cup \bar{U_2} \cup ... \bar{U_m}$ where $m$ is minimal such that $K_1 \subseteq U_1 \cup ... \cup U_m$. Continue so that $K_n = \bar{U_1} \cup ... \cup \bar{U_i}$ and $K_{n+1} = \bar{U_1} \cup ... \cup \bar{U_j}$ where $j$ is minimal such that $K_n \subseteq U_1 \cup ... U_j$. \end{proof} \end{lemma} The same argument can be used to show that for a manifold $M$ whose topology has a countable basis of open sets, there is a sequence of compact subsets $K_1, K_2, ...$ whose union is $M$ and such that $K_1 \subset Int(K_2) \subset ... $ \begin{lemma}\label{lemma:7.5} If $X$ is an open set in $\mathbb{R}^n$, then $X$ can be written as the union of two open sets $U$ and $V$ such that $U, V, $ and $U \cap V$ can be written as the disjoint union of open sets, each of which is a finite union of open rectangles. \begin{proof} Take a sequence of compact sets $K_1 \subset K_2 \subset ...$ as in the above lemma. We will construct the argument using the following: \begin{enumerate} \item[i.] T($U$) is true where $U$ is an open rectangle in $\mathbb{R}^n$ \noindent \textit{Proof.} $H^k(U)$ and $H_k(U)$ vanish for $k>0$, and $H^0(U) = \mathbb{R}$ and $H^\infty_0(U) = \mathbb{Z}$. Then, any element $f \in Hom(\mathbb{Z}, \mathbb{R})$ is determined by $f(1)$, therefore $Hom(\mathbb{Z}, \mathbb{R}) = \mathbb{R}$. So, for an open rectangle $U$ in $\mathbb{R}^n$, $\ell$ is an isomorphism \item[ii.] If $U$ and $V$ are open in a manifold, and if T($U$), T($V$), and T$(U \cap V)$ are true, then T$(U \cup V)$ is true. \noindent \textit{Proof.} We can construct the following maps between Mayer Vietoris sequences for cohomology and homology respectively, and apply the five-lemma. Let $\widetilde{H}^{k-1}(U)$ denote $Hom(H_{k-1}(U), \mathbb{R})$. \begin{center} \begin{tikzcd}[column sep=small, scale=0.8] H^{k-1}(U)\oplus H^{k-1}(V) \arrow[r, ""] \arrow[d, "\ell"] & H^{k-1}(U \cap V) \arrow[r, ""] \arrow[d, "\ell"] & H^k(U \cup V) \arrow[r, ""] \arrow[d, "\ell"] & H^k(U) \oplus H^k(V) \arrow[r, ""] \arrow[d, "\ell"] & H^k(U \cap V) \arrow[d, "\ell"] \\ \widetilde{H}^{k-1}(U) \oplus \widetilde{H}^{k-1}(V) \arrow[r, ""] & \widetilde{H}^{k-1}(U \cap V) \arrow[r, ""] & \widetilde{H}^{k}(U \cup V) \arrow[r, ""] & \widetilde{H}^k(U) \oplus \widetilde{H}^k(V) \arrow[r, ""] & \widetilde{H}^k(U \cap V) \end{tikzcd} \end{center} We know that this diagram commutes by Lemmas (\ref{lemma:comm1}) and (\ref{lemma:comm2}) shown above. Then, we can apply the five lemma (\ref{lm:5}) to see that $\ell:H^k(U \cup V) \rightarrow Hom(H_k(U \cup V), \mathbb{R})$ is an isomorphism. \item[iii.] If $X$ is a disjoint union of open manifolds $X_\alpha$ and each T$(X_\alpha)$ is true, then T$(X)$ is true. \noindent \textit{Proof:} Identifying a class of $H^k(X)$ is the same as identifying a class on each $X_\alpha$. In other words, $H^k(X) = \prod_\alpha H^k(X_\alpha)$. Similarly, $H^\infty_k(X)$ is the same as identifying a class on each $X_\alpha$ such that all but a finite number of these classes are 0. So, $H^\infty_k(X) = \bigoplus_\alpha H^\infty_k(X_\alpha)$. \noindent Then, $Hom(H^\infty_k(X), \mathbb{R}) = Hom(\bigoplus_\alpha H^\infty_k(X_\alpha), \mathbb{R}) = \prod_\alpha Hom(H^\infty_k(X_\alpha), \mathbb{R})$. \end{enumerate} With (i), (ii), and (iii), we can prove $T(X)$ for any smooth manifold $X$: First, we will show it is true when $X\subseteq \mathbb{R}^n$ is the finite union of $p$ open rectangles. Let $U$ be the union of $p-1$ rectangles and $V$ be the other. Then $T(U)$ and $T(V)$ are true by (i) and (iii), and $T(U \cap V)$ is true since $U \cap V$ is also a finite union of rectangles. By (ii), $T(X)$ is true. When $X \subset \mathbb{R}^n$ is an open set, use (\ref{lemma:7.5}) to write $X$ as $U \cup V$ such that $U$, $V$, and $U \cap V$ are disjoint unions of open sets, and each open set is the finite union of open rectangles. Using the previous step and (iii), $T(U)$, $T(V)$, and $T(U \cap V)$ are true, therefore $T(X)$ is true. Since diffeomorphisms between manifolds determine isomorphisms between the cohomology and homology groups, $T(X)$ is true for any manifold diffeomorphic to an open set in $\mathbb{R}^n$. When $X$ is a finite union of $p$ open sets, each diffeomorphic to an open set in $\mathbb{R}^n$, let $U$ be the union of $p-1$ of these sets, and $V$ be the remaining set, and $T(X)$ follows by (ii) and (iii). By (\ref{lemma:7.6}), any smooth manifold $M$ can be written as the union of two open sets $U$ and $V$ such that each $U$, $V$, and $U \cap V$ is a disjoint union of open sets, each of which is diffeomorphic to a finite union of open sets in $\mathbb{R}^n$. By (iii), $T(U)$, $T(V)$ and $T(U \cap V)$ are true. Then, by (ii), $T(M)$ is true. Therefore, for any smooth manifold, $\ell$ is an isomorphism. \end{proof} \begin{ex} Use de Rham's Theorem to find $H^i(S^n)$, $n \geq 1$.
